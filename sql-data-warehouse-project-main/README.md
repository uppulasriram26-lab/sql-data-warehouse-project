# 📊 Data Warehouse & Analytics Project

## 🚀 Overview
This project demonstrates a modern **Data Warehouse and Analytics solution** using a Medallion Architecture (Bronze, Silver, Gold).  
It focuses on building an ETL pipeline, transforming raw data, and creating analytics-ready datasets for reporting and insights.

---

## 🏗️ Architecture

The system follows a layered Medallion Architecture:
📥 Raw Data Sources (CSV / ERP / CRM)
↓
🟤 Bronze Layer (Raw Data Ingestion)
↓
⚪ Silver Layer (Data Cleaning & Transformation)
↓
🟡 Gold Layer (Business-Ready Star Schema)
↓
📊 Analytics & Reporting Layer

---

## 🔄 ETL Process

### 1. Extract
- Data is collected from ERP and CRM CSV sources

### 2. Transform
- Data cleaning (null handling, duplicates removal)
- Standardization of formats
- Data normalization

### 3. Load
- Structured data loaded into SQL Server tables
- Organized into Bronze, Silver, Gold layers

---

## 📊 Data Modeling
- Designed **Star Schema**
- Created Fact and Dimension tables
- Optimized for analytical queries and reporting
