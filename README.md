# 📊 SQL Data Warehouse Project (ETL Pipeline - Bronze, Silver, Gold)

## 📌 Project Overview
This project demonstrates the design and implementation of a modern **data warehouse architecture** using SQL.  
It follows a multi-layer ETL approach (Bronze, Silver, Gold) to transform raw data into analytics-ready datasets.

The goal is to simulate a real-world data engineering workflow used in companies for reporting and analytics.

---

## 🎯 Objective
To build a structured data warehouse pipeline that:
- Ingests raw data (Bronze layer)
- Cleans and transforms data (Silver layer)
- Creates business-ready analytical tables (Gold layer)
