# 📊 SQL Data Warehouse Project (ETL Pipeline - Bronze, Silver, Gold)

## 🚀 Overview
This project demonstrates a modern **data warehouse system** built using SQL Server and follows the **Medallion Architecture (Bronze, Silver, Gold)**.

It simulates a real-world data engineering pipeline used in enterprise environments for analytics and reporting.

---

## 🏗️ Architecture

Data flows through three structured layers:
Raw Data Sources (CSV / ERP / CRM)
↓
🟤 Bronze Layer → Raw ingestion (unaltered data)
↓
⚪ Silver Layer → Data cleaning, transformation, standardization
↓
🟡 Gold Layer → Business-ready star schema for analytics
↓
📊 Reporting & Insights Layer

---

## 🔄 ETL Pipeline

### 🟤 Extract (E)
- Data is collected from multiple sources (ERP & CRM CSV files)

### ⚪ Transform (T)
- Remove duplicates
- Handle missing values
- Standardize formats
- Apply business rules

### 🟡 Load (L)
- Load structured data into SQL Server tables
- Organize into layered architecture (Bronze → Silver → Gold)

---

## 📊 Data Modeling
- Designed a **Star Schema architecture**
- Created Fact and Dimension tables
- Optimized for fast analytical queries
